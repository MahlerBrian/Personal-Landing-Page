Drop your asset files in THIS folder. Vite serves everything here from the
site root ("/"), so the filenames must match what the code references:

  profile.jpg   →  shown in the Hero (referenced as "/profile.jpg")
                   Square image works best (e.g. 400×400). Optional — delete
                   the <img> in src/components/Hero.jsx if you don't want one.

  cv.pdf        →  your resume (referenced as "/cv.pdf" in Hero.jsx and CV.jsx)

You can delete this README.txt once you've added your files.
